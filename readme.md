Tuto installation

npm install 

creez un dossier data ou vous mettez vos db a l interieur

dans config.json ajoutez le token de votre bot, l id du bot et l id du serveur ainsi que l id du role owner bot pour utiliser la commande " delete db " ainsi que les emojies

node index.js pour lancer le bot, les db s importeront automatiquement 

une chose a faire malgré tout, ajouter les index des db dans /utils/deleteWorkerThread.js et /utils/searchWorker.js.
